    wendorPrefixes

    @mixin wendorPrefixes($property, $value) { ... }

    Description

    Helper mixin koji dodaje wendor prefikse zeljenom propertiju kao i vrednost. Dodaje samo "standardne" prefikse -moz, -webkit ... Moguce je dodati sadrzaj u mixin ako je potrebno.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property

    Properti kojem zelimo dodati prefikse

    		Property— none
    $value

    Vrednost prefiksovanog propertija

    		Value— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Prefixing box-size

    @include wendorPrefixes(box-size, border-box);

    Output

    Properti sa dodeljenim prefiksima i vrednoscu

    Used by

    • [mixin] center
    • [mixin] center
    • [mixin] center

    Author

    • Pedja Smigeljski

    center

    @mixin center($position, $offsetLeft: 50%, $offsetTop: 50%) { ... }

    Description

    Apsolutno pozicioniranje elementa unutar roditeljskog kontejnera, koji je li, mora imati position: relative.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $position

    jedno od [vertical, horizontal, both]

    		String— none
    $offsetLeft

    udaljenost od leve ivice kontejnera

    		Value50%
    $offsetTop

    udaljenost od gornje ivice kontejnera

    		Value50%

    Example

    Precizno centriranje elementa unutar roditelja

    @include center(both);

    Requires

    • [mixin] wendorPrefixes
